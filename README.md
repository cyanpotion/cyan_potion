# xenoamess's-civilization
**中文译名：xenoamess的文明**

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

A turn-based civilization strategy game built on cyan_potion engine.

基于 cyan_potion 引擎开发的回合制文明策略游戏。

## Introduction

xenoamess's-civilization is a civilization-style strategy game where players lead a civilization from ancient times to the space age, developing technologies, building cities, engaging in diplomacy, and competing with other civilizations for dominance.

## Features

- Turn-based strategy gameplay
- Multiple civilizations to choose from
- Technology tree spanning from ancient to modern era
- City building and management
- Diplomacy and warfare systems
- Procedurally generated maps

## Current State

This game is currently in early development. Many features are still being implemented.

## Technical Stack

- **Engine**: [cyan_potion](https://github.com/cyanpotion/cyan_potion) - A lightweight 2D game engine in Java
- **Language**: Java
- **Build Tool**: Maven
